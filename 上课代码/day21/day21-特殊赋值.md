## day21-特殊赋值

* 常见赋值方式

  ~~~markdown
  1. a = 10 # 普通变量赋值
  2. 增强赋值
  	a += 10  -= *= /= %=  //=  
  3. 连续赋值
  	a = b = c = 10
  	c = 10
  	b = c  # 变量间赋值  10
  	a = b 
  4. 变量间 交换值
  	a = 10
  	b = 20
  	a,b = b,a
  
  ~~~


* 特殊赋值

~~~markdown
1. = 右边的数据 长度 必须 与 左边 变量的 个数 要一致，否则报错
	a,b = '12'
    a,b = [10,20]
    a,b = (10,20)
    a,b = {1,2}
    a,b = range(2)
    a,b = {'k':10,'v':20}
2. = 左右两侧 数量不对等，也可以 利用切片 进行赋值
	s = 'html'
    a,b = s[0],s[1:]
    print(a)
    print(b)
3. = 左右两侧 数量不对等，使用* 来做 收集
	- *变量 收集 分配完之后剩余的 数据，形成了一个列表
	- *变量 只能出现一次
	- *变量 尽量不要出现 与 普通变量 同名，否则会覆盖
	- 如果当前数据被普通变量接收完毕，则*变量 接收到为空列表
	a,*b = 'html'
    print(a) # 'h'
    print(b) # ['t', 'm', 'l']
~~~
